Cheep - Decentralised social media network

Cheep is a decentralised social network built on the Ethereum blockchain network. The platform is enabled through Solidity smart contracts which ensures Cheepsâ€™ security and decentralised nature.

Architecture

Our smart contract interacts with the browser front-end through web3.js, a collection of libraries which allow you to interact with Ethereum nodes using HTTP, IPC or WebSocket

Interaction with blockchain

Web3.js provides a Javascript interface to Ethereum blockchain nodes, which contain the business logic and data for our application. This library is used for authentication of Ethereum accounts. It also provides a Javascript interface to an Ethereum smart contract, so that user input from the web interface is seamlessly passed into smart contract function calls. MetaMask is a supported browser extension which stores Ethereum wallets and can be used to pay the fee associated with certain smart contract function calls.
